🏠 Property Rental Management System - Static Web

A frontend-only Property Rental Management System built with HTML, CSS, and JavaScript. This is a static web version of an original C# WinForms + SQL Server desktop application, using browser localStorage for data persistence.

✨ Features

1️⃣ Dashboard – Overview of properties, tenants, and key stats
2️⃣ Properties Management – Add, view, and manage rental properties
3️⃣ Tenants Management – Track tenant information and details
4️⃣ Owners Management – Manage property owner records
5️⃣ Agreements – Handle rental/lease agreements
6️⃣ Payments – Track and manage rent payments
7️⃣ Reports – Generate summary reports and insights

🛠️ Tech Stack

🔹 HTML5
🔹 CSS3
🔹 JavaScript (Vanilla)
🔹 Browser localStorage (no backend required)

📂 Project Structure

📄 index.html – Landing/login page
📄 dashboard.html – Main dashboard
📄 properties.html – Properties management
📄 tenants.html – Tenants management
📄 owners.html – Owners management
📄 agreements.html – Rental agreements
📄 payments.html – Payment tracking
📄 reports.html – Reports and analytics
📄 app.js – Core application logic
📄 style.css – Styling

🚀 How to Run

1️⃣ Clone or download this repository
2️⃣ Open index.html in any modern web browser
3️⃣ No installation or server setup required — runs entirely client-side

📝 Notes

This project was originally developed as a full desktop application (C# WinForms with SQL Server, stored procedures, and 22 screens) and later adapted into this lightweight static web version for portability and quick demos.
